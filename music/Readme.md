# Исследование музыкальных предпочтений в городах Москва и Санкт-Петербург

## Данные
* идентификатор пользователя;
* название трека;
* имя исполнителя;
* название жанра;
* город, в котором происходило прослушивание;
* время, в которое пользователь слушал трек;
* день недели.

## Задача
Исследование музыкальных предпочтений в городах Москва и Санкт-Петербург
